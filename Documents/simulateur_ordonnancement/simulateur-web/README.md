# Simulateur d'Ordonnancement de Processus

Simulateur web interactif pour visualiser les algorithmes d'ordonnancement de processus dans un système d'exploitation.

## Algorithmes supportés

- **FIFO** (First In First Out)
- **SJF** (Shortest Job First)
- **Priorité**
- **Round Robin**

## Déploiement

### Option 1 : Serveur web local

```bash
# Utiliser Python
python3 -m http.server 8000

# Ou Node.js
npx http-server
